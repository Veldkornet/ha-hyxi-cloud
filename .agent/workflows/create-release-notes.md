# Creating Premium Release Notes

When the user asks for release notes, always provide them in a carousel format with two slides:

1. **Rendered Preview**: The visual, markdown-rendered version of the notes.
2. **Raw Markdown**: The exact source code within a `markdown` code block for easy copy-pasting.

Example format:
````carousel
# Rendered Title
...content...
<!-- slide -->
```markdown
# Raw Title
...code...
```
````
